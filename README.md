Описание программы
Currency Converter - это GUI-приложение для конвертации валют с использованием актуальных курсов. Приложение подключается к внешнему API для получения курсов валют, сохраняет историю конвертаций и поддерживает работу в офлайн-режиме.

Возможности:
Конвертация между 12 популярными валютами
Получение актуальных курсов через API exchangerate-api.com
Отображение текущих курсов валют
Сохранение истории конвертаций в JSON
Экспорт истории в отдельный файл
Очистка истории
Демо-режим при отсутствии API ключа
Демо-режим при отсутствии интернета
Валидация ввода (положительное число)
Быстрая смена валют местами
Как получить API ключ
Перейдите на сайт https://www.exchangerate-api.com
Нажмите "Get Free Key"
Введите email и зарегистрируйтесь
Скопируйте полученный ключ
При первом запуске программа запросит ключ
Или вставьте ключ в файл api_key.txt
Бесплатный план включает:

1500 запросов в месяц
Обновление курсов раз в сутки
Инструкция по запуску
Требования:
Python 3.6 или выше
Установленная библиотека requests: pip install requests
Tkinter (идёт в комплекте с Python)
Запуск:
Установите зависимости: pip install requests
Перейдите в директорию проекта: cd currency-converter
Запустите приложение: python main.py
Примеры использования
Конвертация валюты:
Выберите исходную валюту (например, USD)
Выберите целевую валюту (например, RUB)
Введите сумму (например, 100)
Нажмите "Конвертировать"
Результат отобразится на экране
Просмотр курсов:
Текущие курсы валют отображаются относительно USD
Курсы обновляются при запуске программы
Работа с историей:
Все конвертации сохраняются автоматически
История отображается в таблице внизу окна
Кнопка "Очистить историю" удаляет все записи
Кнопка "Экспорт в JSON" сохраняет историю в отдельный файл
Тестовые сценарии:
Тест 1: Базовая конвертация
Выберите USD → RUB
Введите 100
Нажмите "Конвертировать"
Проверьте результат в истории
Тест 2: Смена валют местами
Выберите USD → EUR
Нажмите кнопку ⇄
Валюты поменяются на EUR → USD
Тест 3: Валидация
Попробуйте конвертировать с пустой суммой (ошибка)
Попробуйте ввести -100 (ошибка)
Попробуйте ввести буквы вместо цифр (ошибка)
Тест 4: Офлайн режим
Отключите интернет
Запустите приложение
Выполните конвертацию (используются демо-курсы)
Тест 5: История
Выполните 5 конвертаций
Нажмите "Экспорт в JSON"
Закройте программу
Откройте снова - история сохранилась
Валидация данных
Приложение проверяет:

Сумма должна быть положительным числом
Нельзя конвертировать одинаковые валюты
Поле суммы не может быть пустым
Выбранные валюты должны быть доступны
